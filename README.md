# LogicLens - Structure-Aware Code Similarity Detection System

This repository contains the **LogicLens** project for the Data Structures and Algorithms course (Team 1, Sujay S - AIE B 25158).

## About LogicLens

LogicLens is a structure-aware code similarity detection system designed to identify logical resemblance between Python programs without relying on machine learning techniques. The system converts programs into simplified token sequences that represent the underlying logic, removing superficial differences while preserving program structure.
